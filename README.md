# Social Science Data Analysis Tools

A curated academic guide to major quantitative and qualitative data analysis tools used in social science research.

This repository provides:
- Overview of key software
- Recommended academic books and journal references
- Suggested video lectures and interactive courses
- Guidance on methodological positioning

# 1. Quantitative Analysis Tools

## SPSS

Widely used in psychology, education, and applied social sciences for statistical testing and survey analysis.

### Recommended Books
- Field, A. (2018). *Discovering Statistics Using IBM SPSS Statistics.*
- Pallant, J. (2020). *SPSS Survival Manual.*
- George, D. & Mallery, P. (2019). *IBM SPSS Statistics Step by Step.*

### Courses
- LinkedIn Learning: SPSS Essential Training
- Coursera: Logistic Regression with SPSS

---

## Stata

Common in economics, public policy, and applied quantitative research.

### Recommended Books
- Acock, A. (2018). *A Gentle Introduction to Stata.*
- Cameron & Trivedi (2010). *Microeconometrics Using Stata.*
- Rabe-Hesketh & Skrondal (2022). *Multilevel and Longitudinal Modeling Using Stata.*

### Courses
- UCLA IDRE Stata Tutorials
- StataCorp Official Video Series

---

## R

Open-source statistical computing environment with strong reproducibility and visualization capabilities.

### Recommended Books
- Wickham & Grolemund (2017). *R for Data Science.*
- Dalgaard (2008). *Introductory Statistics with R.*
- Fox & Weisberg (2019). *An R Companion to Applied Regression.*

### Courses
- Coursera: Data Analysis with R
- RStudio Cheatsheets & Tutorials

---

## Python

Used in computational social science, machine learning, and natural language processing.

### Recommended Books
- Grus (2019). *Data Science from Scratch.*
- McKinney (2022). *Python for Data Analysis.*
- Géron (2022). *Hands-On Machine Learning with Scikit-Learn.*

### Courses
- Coursera: Python for Everybody
- Kaggle Micro-Courses

---

## Mplus

Specialised software for structural equation modeling, latent class analysis, and advanced modeling.

### Recommended Books
- Muthén & Muthén. *Mplus User's Guide.*
- Kline (2016). *Principles and Practice of Structural Equation Modeling.*
- Wang & Wang (2020). *Structural Equation Modeling.*

### Tutorials
- UCLA IDRE Mplus Resources

# 2. Qualitative Analysis Tools

## NVivo

Supports coding and thematic analysis of interview transcripts, policy texts, and mixed-methods research.

### Recommended Books
- Bazeley & Jackson (2013). *Qualitative Data Analysis with NVivo.*
- Lewins & Silver (2014). *Using Software in Qualitative Research.*

### Tutorials
- University Library NVivo Workshops
- Official NVivo Training Videos

---

## MAXQDA

Qualitative and mixed-methods software integrating coding and quantitative comparison.

### Recommended Books
- Kuckartz & Rädiker (2019). *Analyzing Qualitative Data with MAXQDA.*
- Saldaña (2021). *The Coding Manual for Qualitative Researchers.*

### Tutorials
- MAXQDA Official YouTube Channel

---

# 3. Choosing the Right Tool

| Research Need | Recommended Tool |
|---------------|------------------|
| Basic hypothesis testing | SPSS |
| Policy evaluation & panel data | Stata |
| Reproducible academic research | R |
| Machine learning & NLP | Python |
| Structural Equation Modeling | Mplus |
| Interview-based research | NVivo |
| Mixed-methods design | MAXQDA |
